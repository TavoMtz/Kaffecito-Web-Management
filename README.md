# ☕ Coffee Shop Management System (Kaffecito)

This is a comprehensive web system designed for the management and operation of a local coffee shop. It connects the owner, employees, and customers within a unified, secure platform.

## Tech Stack
* **Backend:** PHP
* **Frontend:** JavaScript, HTML5, CSS3
* **Database:** SQL / MySQL

## Core Features

### Admin Dashboard (Owner)
* **Inventory Management:** Complete control over the product catalog, allowing creating, updating, and deleting items.
* **User Management:** Ability to assign specific roles and privileges to workers.
* **Reporting:** Visual tracking and insights into business activities and performance.

### Employee System
* **Order Tracking:** Optimized interface for workers to process, update, and manage incoming orders efficiently.
* **Segmented Privileges:** Restricted access based on the role assigned by the administration to secure data integrity.

### Customer Experience
* **User Authentication:** Customers can create accounts to access exclusive benefits and track their order history.
* **Customized Promotions:** Access to special offers and targeted discounts for registered users.
* **Order-Ahead Flow:** Guest and account-based checkout system tailored for in-store pickup.
